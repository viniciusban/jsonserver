JSON SERVER
===========

Get a fake REST API with zero coding in seconds.

Inspired by https://github.com/typicode/json-server

THIS IS A WORK IN PROGRESS.

Usage
-----

```
$ python jsonserver.py
```

There are 2 related objects: `careers` and `people`.

Valid URL's:

- /people
- /people/1
- /careers/1/people
- /careers/add
- /careers/1/edit
